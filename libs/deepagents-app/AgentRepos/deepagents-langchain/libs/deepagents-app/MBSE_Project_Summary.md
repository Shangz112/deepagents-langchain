## 使用指南

### 1. 导入MagicDraw
1. 启动MagicDraw System Engineers 2022x
2. 选择"File" → "Import" → "XMI"
3. 选择`SatelliteSolarArray_MBSE_Model.xmi`文件
4. 确认导入设置并完成导入

### 2. 智能体系统启动
```bash
# 启动智能体辅助系统
python satellite_solar_array_ai_assistant.py

# 或加载特定MBSE模型
assistant = SatelliteSolarArrayAIAssistant("satellitesolararraymaintenancetest_project.json")
```

### 3. 运维操作流程
1. **开始操作**: `assistant.start_operation("deployment", "operator_id")`
2. **执行步骤**: `assistant.execute_step("DEPLOYMENT-001", parameters)`
3. **监控状态**: `assistant.detect_anomaly(sensor_data)`
4. **合规检查**: `assistant.check_compliance()`
5. **生成报告**: `assistant.generate_report()`
